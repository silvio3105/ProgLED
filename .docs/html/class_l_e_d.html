uint8_t LED::config = (1 << PROG_LED_STATUS_BIT) | (100 << PROG_LED_BRGHT_BIT)
private

LED config variable.

First seven bits are reserved for LED brightness. Last bit(MSB) is reserved for toggling LED. It is possible to turn off or on LED without changing LED's color.
