◆ chBitmap

uint8_t LED::chBitmap = 0
private

Color channel bitmap variable.

Variable format is 0b00BBGGRR where RR bits represent channel index for red color, GG bits represent channel index for green color and BB bits represent channel index for blue color. Variable is set via setChannelBitmap method and color channel index is fetched via getChannelIdx method.

◆ config

uint8_t LED::config = (1 << PROG_LED_STATUS_BIT) | (100 << PROG_LED_BRGHT_BIT)
private

LED config variable.

First seven bits are reserved for LED brightness. Last bit(MSB) is reserved for toggling LED. It is possible to turn off or on LED without changing LED's color.
